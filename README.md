# color-jump
ColorJump é um jogo arcade viciante onde guias uma esfera néon através de obstáculos com simples toques na tela. Conforme pontuas, o jogo acelera e o cenário transforma-se dinamicamente entre o Nascer do Sol, Dia, Pôr do Sol e Noite.  Supera o teu recorde e testa os teus reflexos neste desafio rápido e eletrizante!
